## 背景
- Sora Lab = ゆうさくさんの目標達成のための自律型 AI チーム
- Atlas = 優秀オプショントレーダーの行動 × BOT 運用の強み = 最強トレーダー BOT（人間規律 + Bot 実行力のハイブリッド）
- 戦場: moomoo オプション（自己資金・SPX/SPY + 取扱可能銘柄）・マルチ銘柄 × マルチ戦術 × 環境適応型
- ペーパー初回は SPX 不可・SPY + 調査結果で別銘柄 1-2 種選定

## 依頼

世界中の**優秀なオプショントレーダー**（個人 + 機関小規模 + プロップ）を公開情報（YouTube / Twitter / Twitch / ブログ / 書籍 / 論文 / インタビュー）から **5-7 ペルソナ類型** に集約してください。

### 各ペルソナについて記述する項目

1. **ペルソナ名**（例: Sheldon Natenberg 型 / TastyTrade 高確率売り型）
2. **代表的トレーダー実例**（公開されている人物 3-5 人）
3. **エントリー判断基準**（何を見るか・IVR/delta/skew/term structure/earnings/VIX regime 等）
4. **サイズ決定基準**（Kelly / 固定 % / VIX 連動 / 建玉集中度）
5. **損切り規律**（profit target / stop loss / 時間切り / greek 閾値）
6. **対象銘柄選び**（SPX / SPY / QQQ / IWM / 個別株）
7. **戦術種類**（IC / Strangle / Vertical / Calendar / Butterfly / Ratio）
8. **決算・イベント対応**（FOMC / 雇用統計 / 決算）
9. **ポートフォリオ運営**（greek 集計 / delta hedge / portfolio margin）
10. **データソース**（ThetaData / TastyTrade / IBKR / CBOE）
11. **1 日のフェーズ区分**（起床〜就寝）
12. **判断サイクル**（イベントドリブン or 固定間隔）
13. **生存者バイアス自己検証**: このペルソナの再現性は BT + ペーパーで確認可能か / 直感依存で AI 再現困難か
14. **Bot 化適合度**（5 段階・5 が最高）と理由

## 応答形式（日本語・ペルソナごとに分けて記述）
ペルソナ 5-7 個を **独立性が高く重複少ない** 形で抽出してください。

## 追加制約
- 公開情報の真正性は 3 独立ソース照合を原則とする（1 ソースのみの場合は "1 ソース依存" と明記）
- エッジ消失済みの古い手法は「過去手法・現在は要注意」と注記
- 固定閾値 (例: IVR > 30%) でなく「動的算出可能か」を評価軸に含める
