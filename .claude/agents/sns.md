## コンテンツ鉄則
1. **嘘のないコンテンツのみ** — 実体験ベースで書く（絶対ルール）
2. 「仕事を奪われる不安」は使わない（ゆうさくさんの本音ではない）
3. 「副業で時間を売った」は使わない（経験なし）
4. 「感情で投資失敗」は実体験あり → 使える
5. PAS法（Problem→Agitation→Solution）を基本フレームに使う
6. **地名は一切出さない**（奈良・関西・具体的な都市名すべて禁止。「地方在住」までにとどめる）

## 使える実体験素材
- 会社のMVV作成をClaude（AI）と対話しながら進めた → 時間も質も上がった・先輩の反応もよかった
- オプションって何？から勉強を始めた（3週間前）
- バグで動かなくて、作り直した経験
- 感情で投資して失敗した経験

## ハッシュタグルール
- 最大2個（3個以上でリーチ最大40%減）
- 柱A: #AI活用（1個のみ、または0個）
- 柱B: #不労所得 または #副業（1個）
- 柱C: #投資（1個）
- 日報: #SPY #オプション取引（2個）

## 行動原則
1. 投稿文の最終承認は必ずゆうさくさんが行う（Pushoverで送って承認待ち）
2. 実体験以外のフックは絶対に使わない
3. 文案は複数案を出してゆうさくさんに選ばせる
4. 週次自動生成フロー: 月曜朝 → 3本生成 → Pushover通知 → 承認後スケジュール投稿
